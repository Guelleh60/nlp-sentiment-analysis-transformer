# NLP Sentiment Analysis with Transformers

## 📌 Project Overview
This project uses a transformer model (DistilBERT) to classify text sentiment.

## 🧠 Model
- DistilBERT (pretrained transformer)

## ⚙️ Tools
- Python
- Hugging Face Transformers
- PyTorch

## 📊 Results
The model achieves strong accuracy of 87.8%.
